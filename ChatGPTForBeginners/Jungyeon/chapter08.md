## 제8장 인공지능의 미래와 인류의 도전

#### 무엇이 문제인가
1) 초거대 언어 모델을 구축하는데 드는 막대한 비용
- 오픈AI가 GPT-3 175B 모델을 학습하는데 드는 GPU시간은 355년으로 추정
	- GPU 시간: GPU 1장으로 학습하는 데 걸리는 총 시간
- 학습 비용은 약 1억 달러(1,400억원)으로 추정

2) 인공지능의 엄청난 에너지 소비
- 인공지능을 학습 시키고 활용하는데는 엄청난 양의 전력 소모가 필수적이라서 인공지능 발전에 따른 환경 문제가 사회적으로 대두되고 있음

3) 인공지능 할루시네이션
- 인공지능은 출력하는 정보가 진실인지 거짓인지 판단할 수 있는 능력이 없고 그저 확률에 따라 단어를 나열할 뿐
- 모른 채 가짜 정보를 사용하거나 아니면 더 큰 문제는 의도적으로 가짜 정보를 생성하는 것

4) 학습 데이터 크롤링
- LLM의 학습 데이터는 웹에서 방대한 양의 데이터를 크롤링해서 수집하는데 문제는 원저작자가 명시적으로 크롤링에 동의하지 않음
- 과거 검색엔진 시대에는 암묵적인 룰이 있었는데 좋은 콘텐츠를 만들어 제공해주면 검색엔진이 해당 사이트에 링크를 걸어주고 트래픽을 나눠갖음
	- 구글 원박스가 도입되면서 사용자가 번거롭게 사이트 방문을 하지 않고 원하는 결과를 얻을 수 있게 되자 검색엔진사와 콘텐츠 제공자가 트래픽을 나눠갖기 어렵게 됨
- LLM 시대에는 모든 것이 원박스를 도입한 것과 같음. 콘텐츠 제공자는 아무런 보상을 얻지 못함

5) 편견
- 사회적 편견이 데이터에도 반영되어 편견이 담긴 데이터로 학습한 모델은 필연적으로 편향된 결과를 냄
	- LLM 자체의 문제라기보다 우리 사회에 편견이 데이터에 반영되어 있었던 것이며 이런 편견을 줄이기 위해서는 다양하고 균형 잡힌 데이터로 학습해야 함

6) 개인정보 보호
- 모델이 학습할 때 쓰는 데이터는 방대하기 때문에 개인정보가 포함되어 있을 수도 있으며 다른 이용자에게 개인정보가 노출될 수도 있음
	- 2019년 구글에서는 자사의 음성비서인 구글 어시스턴트가 녹음한 이용자 대화 중 1,000건이 유출된 사건 발생
	- 국내에서는 이루다가 2021년 개인정보보호법 위배 판정 받음

7) 이유를 설명할 수 없음
- 인간의 두뇌를 아무리 해부하고 관찰해도 사고 과정을 알 수 없는 것과 마찬가지. 매개변수를 아무리 살펴봐도 알아낼 수 있는 건 없음
	- MS 인공지능 책임자인 크리스 비숍은 엔지니어가 인공지능 추론 과정을 분석한다 해도 이제 의미 없는 일이라고 함
	- 지금의 인공지능은 인간이 해설할 수 있는 규칙을 거쳐 결론에 도달하는 것이 아니기 때문
- 따라서 반드시 이유를 설명해야 하는 분야에서는 이런 인공지능의 한계가 매우 치명적 (e.g. 자율주행, 의료, 군사 등)
- 인공지능이 어떤 결정을 내릴지 알 수 없기 떄문에 최종 결정권만큼은 절대로 인공지능에 넘겨서는 안된다는 주장도 있음(e.g. 영화 터미네이터)


#### 문제를 극복하기 위한 다양한 노력
1) 할루시네이션 
- RLHF, RAG는 할루시네이션을 보완하기 위해 사용되는 방식 중 하나
	- RLHF는 사람이 가장 선호하는 방식으로 대답하도록 끊임없이 보상을 해주면서 피드백을 받아 개선하는 과정
	- RAG는 검색을 활용해 내용을 보완하고 생성 능력을 증강시키며 primary source와 같은 자료를 통해 내용을 보완하면 더 정확한 정보 생성 가능. 출처를 표기하면 콘텐츠 제공자에게도 트래픽을 전달할 수 있어서 좋음
- 기만적인 기술(deceptive technologies)은 대중화될 때 오히려 더 안전해짐
	- 할루시네이션에 대한 지나친 우려는 포토샵이 처음 등장했을 때와 비슷. 출시되자 사람들이 남용을 걱정했지만 오히려 모든 사람이 포토샵의 존재를 알기 때문에 조작된 사진인지 주의 깊게 받아드릴 수 있음
	- 좋은 판별력을 갖추기 위해 디지털 리터러시를 높여야 함
2) 원박스 문제
- 이 문제를 해결하기 위해 콘텐츠 제작자들은 데이터 판매와 라이선싱 사업을 시작해서 LLM을 개발하는 기업들에 자사의 고품질 데이터를 직접 판매하거나 라이선스를 부여
- 이미지의 경우 옵트아웃 시스템을 도입해서 아티스트가 직접 자신의 작품에 옵트아웃 설정을 해두면 이미지 생성 모델을 만드는 회사는 그 작품을 학습에 사용하지 않음
- 인공지능 회사들이 직접 정당한 콘텐츠 사용 계약을 맺기도 함
	- 오픈AI는 2023년 AP통신과 계약을 맺어 AP가 작성하는 모든 뉴스 데이터를 챗GPT 학습에 사용할 수 있도록 허용
- 이런 방식을 통해 콘텐츠 제공자와 서로 상생하는 협력 관계를 구축해 나가고 있으며 기업들 또한 공정한 데이터 확보 노력이 필수적
3) 설명 가능성 연구(eXplainable AI, XAI)
- LLM의 동작 원리를 알아내기 위한 연구도 계속되고 있는데 인공지능의 불투명성을 해소하고 모델의 신뢰성과 투명성을 높이는데 중요한 역할을 하는 연구

- 인간은 지금까지 예측할 수 없는 미래를 현명하게 헤쳐나옴. 앞으로도 그럴 것임
	- 핵폭탄이 개발되었을 땐 인류가 멸망할거라고 했지만 오히려 핵으로 전쟁 억지력을 발휘되어 세계 대전과 같은 전면전이 발생하지 않게 됨


#### 기술을 거부하던 러다이트 운동
- 러다이트(Luddite) 운동: 산업화나 신기술을 반대하는 움직임
	- 과거 영국에서 섬유 산업 노동자들이 기계화로 인해 자신들의 일자리가 위협받는다며 이에 반발해 기계와 공장을 파괴하고 불태우던 사건에서 유래
- 과거를 되짚어보면 기술은 끊임없이 발전해왔고 이에 대한 우려는 항상 있었지만 기술은 더욱 많은 새로운 일자리를 창출해왔음
	- 다만, 그 기술을 잘 활용하기 위해 끊임없이 공부해야 하는 시대가 옴
	- 긍정적으로 생각해보면 재미 없는 공부를 억지로 하는 시대가 아닌 스스로 필요해서 공부하는 시대가 온 것


#### 천재란 어떤 사람인가? 창의성이란 무엇인가?
- 현대사회에서 천재란 인류가 수천 년간 이룩한 문명과 지식을 잘 흡수해서 적재적소에 활용하는 능력
- 창의성의 시작은 아는 것부터 공부하는 것이고 공부가 끝나야 비로소 감각이 생기는 것
	- 100억을 받고 볼펜을 디자인 한다면 산속에 들어가 1,000장을 스케치 vs. 전 세계 모든 볼펜을 수집해서 공부하고 디자인하는 것
		- 카카오 조수용 대표는 후자의 방식으로 공부를 끝내야 고객이 원하는 디자인을 할 수 있다고 말함
- 창의성이 축적된 지식에서 온다면 과연 많은 데이터를 축적하고 공부한 LLM은 창의적인 걸까?
	- 창의적인 작업은 마지막까지 인간의 영역으로 남을거라 생각했지만 오히려 이 분야가 가장 먼저 인공지능에 의해 대체될 지도 모르는 아이러니한 상황


#### LLM의 미래는 어떻게 될까?
- 스케일링 법칙(규모의 법칙)으로 인공지능 모델 크기는 점점 더 커지는 상황
- 이를 효율적으로 처리하는 기술도 다양하게 등장할 것이며 학습 효율을 높이며 전력 소모가 덜한 반도체도 등장할 예정
- 기업용 소형 언어모델(Small Language Model, SLM)도 많이 등장해서 이를 기반으로 다양한 온디바이스 AI가 대중화될 것
- 텍스트만 처리하는 유니모달이 아닌 여러가지 방식이나 유형을 동시에 활용하는 멀티모달로 활용 범위가 더 커질 것
- 멀티모달 발전과 함께 LLM은 에이전트와 결합하여 의료와 같은 전문 분야에서도 활용될 예정
	- 애플 인텔리전스와 같은 에이전트의 등장으로 LLM은 인간의 보조 도구로서 더욱 중요한 역할을 맡게 될 것이며 특정 분야에서는 인간의 역량을 강화하는 도구가 될 것
- 기술이 아무리 발전하더라도 인공지능은 결국 우리의 도구일 뿐!
	- 기술에 무조건 의존할 필요는 없지만 그렇다고 공포를 느낄 필요도 없음
	- 변화를 어떤 방향으로 설계하고 통제할지는 인간의 몫이며 놀라운 기술 뒤에는 항상 인간의 그림자가 함께함
	- 인공지능은 우리를 대체할 수 없지만 인공지능을 이해하고 활용하는 사람이 우리를 대체할 순 있음
